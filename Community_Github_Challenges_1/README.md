# Highest Digit *Challenge 1*

Create a function that receives a number as an argument and returns the highest digit in that number.



### Example
- highestDigit(12899) -> 9
- highestDigit(8123) -> 8
- highestDigit(193) -> 9
- highestDigit(1)-> 1
